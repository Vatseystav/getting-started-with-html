HTML = HyperText Markup Language represented by hyperlinks
Invented in early 1990s, currently consists of 30 billion pages.

Stucture of the Web

You/User -> Internet -> Servers ---- and reversed

Your browser (Firefox, Chrome) sends a requst represented as HTTP (browser language).
Browser makes request to the server
Server in turn honors request by sending HTML back to browser which represents your webpage request.

INLINE VS. BLOCK

HTML language is either inline or block.
Inline language is usually all on one line such as bold.
Block elements/language has content and therefore the browser creates an invisible box around the element.
As an example paragraph is a block element that creates this box and normally start and end with a new line.
Span - Inline ---- Div - Block
Span and Div contain the their content. Div creates a box of text
